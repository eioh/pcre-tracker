# プリコネR ゲーム仕様リファレンス（このプロジェクト向け）

最終更新日: 2026-02-23

このドキュメントは、`pcre-tracker` が扱うプリコネR育成情報を、
「公式Web情報」と「このリポジトリの実装値」の両方から整理したものです。

## 1. 用語

### 専用1（キャラ専用装備1）とは
- 各キャラに設定される専用装備。
- 作成に「キャラのメモリーピース」「プリンセスハート」「マナ」が必要。
- 公式ヘルプ上は、上限解放でメモリーピースとプリンセスハートを使う育成項目として説明される。

このプロジェクトでの対応項目:
- 専用1レベル（`0, 30, 50, ... , 370`）
- 専用1SP装備状態（`ue1SpEquipped`）

### 専用1 SPとは
- 2026-02-15時点の公式お知らせで、コンテンツ名として「キャラ専用装備1SP」が明記されている。
- 本プロジェクトでは、専用1の最終段階として `SP` を管理する。

このプロジェクト実装上の扱い:
- 専用1レベル370到達後、SP化に追加素材が必要という前提で計算。
- 追加コストはメモリーピース `300` を固定値として採用。

### 専用2（キャラ専用装備2）とは
- 装備と強化に「同名キャラのピュアメモリーピース」「マナ」が必要。
- 同名であれば衣装違いキャラのピュアメモリーピースも使用可能。

このプロジェクトでの対応項目:
- 専用2レベル（`0, 1, 2, 3, 4, 5`）

### 才能開花（星上げ）
- ☆を上げる育成。
- 本プロジェクトは ☆1〜☆6 を管理。
- ☆6はキャラごとの実装有無をマスターで判定している（`implemented.star6`）。

### 限界突破
- 本プロジェクトでは ON/OFF の進捗として管理。
- 必要メモピは「通常120 / 限定40」の固定値として扱う。

## 2. 専用1を上げるのに必要な素材（このプロジェクト実装値）

参照実装:
- `src/utils/ue1MemoryCost.ts`
- `src/utils/ue1HeartFragmentCost.ts`
- `src/domain/levels.ts`

### 2.1 専用1メモリーピース必要数

レベル遷移ごとの追加必要数:

| 現在Lv | 次段階までのメモピ |
|---|---:|
| 0 | 50 |
| 30, 50, 70, 90, 110 | 各10 |
| 130〜220 | 各15 |
| 230〜360 | 各5 |
| 370→SP | 300 |

補足:
- Lv0からLv370までの合計は `320`。
- Lv0からSPまでの合計は `620`（= 320 + 300）。
- `implemented_max` モード: 専用1未実装キャラは0扱い。
- `sp_max` モード: 専用1未実装キャラも将来実装想定でSPまで計算。

### 2.2 専用1プリンセスハート（欠片）必要数

レベル遷移ごとの追加必要数:

| 現在Lv | 次段階までのハート欠片 |
|---|---:|
| 0 | 30 |
| 30, 50 | 各5 |
| 70 | 8 |
| 90〜350 | 各10 |
| 360→370 | 20 |

補足:
- Lv0からLv370までの合計は `318`。
- SP装備済みなら必要数0として扱う。
- `implemented_max` モード: 専用1実装キャラのみ対象。
- `all_max` モード: 専用1未実装キャラも将来実装想定で集計。

## 3. 星上げ・限界突破の必要メモピ（このプロジェクト実装値）

参照実装:
- `src/utils/starMemoryCost.ts`
- `src/utils/limitBreakMemoryCost.ts`

### 3.1 星上げ必要メモピ

| 遷移 | 通常キャラ | 限定キャラ |
|---|---:|---:|
| ☆1→☆2 | 30 | 30 |
| ☆2→☆3 | 100 | 100 |
| ☆3→☆4 | 120 | 120 |
| ☆4→☆5 | 150 | 150 |
| ☆5→☆6 | 50 | 25 |

合計:
- ☆1→☆6（通常）: `450`
- ☆1→☆6（限定）: `425`

### 3.2 限界突破必要メモピ

- 通常キャラ: `120`
- 限定キャラ: `40`
- 既に限界突破済みなら `0`

## 4. 最新性メモ（2026-02-23時点）

- 公式アップデート（2025-11-14公開）で、専用1強化上限は `360→370` に引き上げ済み。
- 同お知らせで、`360→370` の解放時に必要なプリンセスハート（欠片）が `20個` と明記されており、実装値と一致。
- 公式お知らせ（2026-02-15公開）で「キャラ専用装備1SP」という名称が登場済み。

- コネクトオーブ・アーツ
- コネクトオーブ・ソウル
